# getTableName

Returns the table name of the passed in entity

## Returns

* This function returns _string_

## Arguments

| Key    | Type   | Required | Default | Description |
| ------ | ------ | -------- | ------- | ----------- |
| entity | string | Yes      | ---     |             |

## Examples

```javascript
var persistedTable = ormService.getTableName( "Category" );
```
