# getPropertyNames

Returns the persisted Property Names of the entity in array format

## Returns

* This function returns _array_

## Arguments

| Key    | Type   | Required | Default | Description |
| ------ | ------ | -------- | ------- | ----------- |
| entity | string | Yes      | ---     |             |

## Examples

```javascript
var properties = ormService.getPropertyNames("User");
```
